
### Installation
01. `git clone `
02. `cd professeurs`
03. `composer install`
04. `npm install`
05. `cp .env.example .env`
06. `php artisan key:generate`
07. `php artisan migrate`
08. `php artisan db:seed`
09. `php artisan serve` Si vous utilisez Laragon pas necessaire.
